### 2. **Defining Fixed Variables for Ionic Radii**

The Colvars module supports fixed variables that can be used in expressions:

```tcl
variable r_A 1.6
variable r_B 0.7
variable r_X 1.4
```

Then modify your customFunction:

```tcl
customFunction {
    name tolerance_ratio_func
    expression "(${r_A} + ${r_X}) / (sqrt(2)*(${r_B} + ${r_X}))"
}
```

Make sure these variables are defined in the same `colvars.conf` or passed appropriately.
